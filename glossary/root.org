Root (raiz/origem)  

  

O root (ou super usuário) é um tipo de usuário de sistemas operacionais baseados em Unix que possui permissões especiais para fazer qualquer operação sobre o sistema e tem acesso a todos seus processos e arquivos.  Nesses sistemas, qualquer usuário com identificador (UID) 0 é um super usuário.
